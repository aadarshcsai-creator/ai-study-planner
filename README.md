# AI Study Planner (CLI Tool)

## 📌 Problem
Students often struggle to manage their study time efficiently, leading to poor productivity and stress.

## 💡 Solution
This project provides a command-line tool that generates an optimized study schedule based on subject priorities and available study hours.

## ⚙️ Features
- Priority-based subject allocation
- Time optimization
- Simple and fast execution
- Output saved to file

## 🛠️ Technologies Used
- Python
- Command Line Interface (CLI)

## 📥 Installation
Clone the repository:
git clone https://github.com/aadarshcsai-creator/ai-study-planner.git

Navigate to folder:
cd ai-study-planner

## ▶️ Run the Project
python main.py

## 🧪 Example
Input:
Subjects: Math (3), Physics (2), Programming (5)  
Hours: 8  

Output:
Math: 2.4 hours  
Physics: 1.6 hours  
Programming: 4.0 hours  

## 📂 Output File
The generated schedule is saved in:
sample_output.txt

## 📌 Note
Ensure Python is installed before running the project.
